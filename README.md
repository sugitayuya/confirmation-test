## 環境構築

Dockerビルド

　1. gitリンク：https://github.com/sugitayuya/confirmation-test
 
 2. docker-compose up -d -build


Laravel環境構築

1. docker-compose exec php bash
   
2. composer install
  
3. php artisan key:generate
   
4. php artisan migrate
